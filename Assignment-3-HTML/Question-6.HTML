Differences Between HTML and HTML5

HTML HTML5
Used for creating basic web structure. Includes advanced features like audio, video, and canvas support.
Fewer semantic elements (e.g., <div> used extensively). Introduces semantic tags like <article>, <section>, <nav>.
No native support for audio and video elements. Provides native support using <audio> and <video> tags.
Requires third-party plugins for animations and graphics (e.g., Flash). Includes <canvas> and SVG for drawing graphics natively.
Form controls and validations are limited and require JavaScript. Includes new input types and built-in form validation (e.g., email, telephone).
