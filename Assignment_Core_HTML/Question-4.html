Common HTML Tags

  1. <html> – Root element that wraps the entire HTML document.
  2. <head> – Contains meta-information like title, styles, and scripts.
  3. <title> – Sets the title of the web page shown in the browser tab.
  4. <body> – Contains the visible content of the web page.
  5. <h1> to <h6> – Define headings, where h1 is the highest level and h6 the lowest.
  6. <p> – Represents a paragraph of text.
  7. <a> – Defines a hyperlink to another page or URL.
  8. <img> – Embeds an image into the page.
  9. <ul> – Creates an unordered (bulleted) list.
  10. <ol> – Creates an ordered (numbered) list.
  11. <li> – Defines a list item inside <ul> or <ol>.
  12. <div> – Generic container used to group elements for styling or layout.
  13. <span> – Inline container used to style small parts of text.
  14. <form> – Defines a form to collect user input.
  15. <input> – Represents a field where users can enter data.
  16. <button> – Creates a clickable button.
  17. <table> – Used to create a table structure.
  18. <tr> – Defines a row in a table.
  19. <td> – Defines a cell in a table row.
  20. <br> – Inserts a line break in text.
